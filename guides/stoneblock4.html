Learn how to install StoneBlock 4 via CurseForge in Prism Launcher

StoneBlock 4 on CurseForge

Official modpack from FTB/CurseForge

View on CurseForge

1 Install Prism Launcher

If you haven't already, install Prism Launcher first.

  1. Visit prismlauncher.org/download
  2. Download the installer for your operating system
  3. Run the installer and follow the on-screen instructions
  4. Launch Prism Launcher and log in with your Microsoft account

Screenshot: Prism Launcher main window

üí° Already have Prism? Skip to Step 2!

2 Enable CurseForge in Prism Launcher

Before importing CurseForge modpacks, you may need to configure API access.

  1. In Prism Launcher, go to Settings (gear icon)
  2. Navigate to APIs in the left sidebar
  3. Under CurseForge, you have two options:
    • Option A: Get a free API key from CurseForge Console (recommended)
    • Option B: Leave it blank (will work for most modpacks)
  4. Click Close to save settings

Screenshot: API settings in Prism Launcher

3 Add StoneBlock 4 via CurseForge

Now we'll import StoneBlock 4 directly from CurseForge.

  1. Click "Add Instance" (the + button in the top-left)
  2. Select "CurseForge" from the left sidebar
  3. In the search box, type "StoneBlock 4" or "FTB StoneBlock 4"
  4. Find FTB StoneBlock 4 in the results and click on it
  5. Choose the version that matches the server (usually the latest)
  6. Click "OK" to start downloading

Screenshot: CurseForge search showing StoneBlock 4

‚ö†Ô∏è Be Patient: StoneBlock 4 is a large modpack (~200+ mods). Download may take 5-15 minutes depending on your internet speed.

4 Launch and Connect to Server

Once the download completes, you're ready to play!

  1. Double-click the StoneBlock 4 instance to launch
  2. Wait for Minecraft to fully load (first launch takes a while due to mod loading)
  3. Click "Multiplayer"
  4. Click "Add Server"
  5. Enter the server address from the Game Servers page
  6. Click "Done" and then "Join Server"

Screenshot: StoneBlock 4 main menu

üéâ You're all set! Enjoy mining through stone in StoneBlock 4!

‚öôÔ∏è Recommended Memory Settings

StoneBlock 4 needs more RAM than vanilla Minecraft. Here's how to increase it:

  1. Right-click the StoneBlock 4 instance
  2. Select "Edit"
  3. Go to "Settings" in the sidebar
  4. Check "Memory" to enable custom settings
  5. Set Maximum memory to:
    • 6144 MB (6GB) - Minimum recommended
    • 8192 MB (8GB) - Optimal for smooth gameplay
  6. Click "Close" to save

Screenshot: Memory settings in instance configuration

‚ö†Ô∏è Not Enough RAM? If you only have 8GB of system RAM, allocate no more than 4-5GB to Minecraft. Leave room for your OS!

üîß Troubleshooting

Game crashes on launch?

  • Make sure you have Java 17 or higher installed
  • Increase allocated RAM to at least 6GB
  • Try updating your graphics drivers

CurseForge modpack not showing up?

  • Try getting a free API key from CurseForge Console
  • Make sure you're connected to the internet
  • Try searching for "FTB StoneBlock 4" instead of just "StoneBlock"

Can't connect to server?

  • Double-check the server address on the Game Servers page
  • Make sure your modpack version matches the server version
  • Check if the server is online (look at the status indicator)
